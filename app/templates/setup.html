Set up your subjects

Add the Google Calendar links for each of your IITM courses. You can limit which events are fetched by setting a date range.

Where to find the calendar link:
Go to your IITM course portal → find the session calendar → click the embed link or share URL.
It usually looks like:
https://calendar.google.com/calendar/embed?src=c_xxxxx%40group.calendar.google.com&ctz=Asia%2FKolkata

Time period (optional): Leave blank to fetch all events, or set a start and end date to only see lectures within that window.
